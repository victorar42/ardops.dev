# Design System

## Tipografía
- **Headings y código:** JetBrains Mono (Google Fonts, self-hosted)
- **Body:** Outfit (Google Fonts, self-hosted)
- Escala: 0.875rem / 1rem / 1.25rem / 1.5rem / 2rem / 3rem

## Paleta de colores (dark terminal)
- `--bg-primary: #0a0a0a`
- `--bg-elevated: #141414`
- `--text-primary: #e4e4e7`
- `--text-muted: #71717a`
- `--accent: #00ff9f` (terminal green)
- `--accent-hover: #00cc7f`
- `--border: #27272a`
- `--error: #ef4444`
- `--warning: #f59e0b`

## Espaciado (escala 4px)
- 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 / 128

## Breakpoints
- Mobile: < 640px
- Tablet: 640-1024px
- Desktop: > 1024px

## Componentes principales
- Hero con animación
- Talk card
- Pipeline visualization (7 stages)
- Blog post card
- Code block (con syntax highlighting)
- Contact form

## Dark mode
- Único modo soportado (terminal aesthetic)
- No hay toggle a light mode

- Sistema único: [definir, ej: Lucide, Heroicons, custom SVG]
