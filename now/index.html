Now — qué estoy haciendo este mes

Inspirado en el movimiento nownownow.com: una sola página, actualizada cada 4-6 semanas, que resume en qué ando ahora mismo. Más concreto que un blog, más honesto que un perfil estático.

// trabajo

Trabajo

  • Construyendo ardops.dev bajo Spec-Driven Development con GitHub Spec Kit: cada feature pasa por /specify → /plan → /tasks → /implement antes de tocar HTML.
  • Diseñando pipelines GitHub Actions con gates de seguridad por defecto (CSP, anti-tabnabbing, JSON-LD, a11y, lighthouse) reutilizables para clientes de banca.
  • Mentoreando a un equipo en Costa Rica sobre DevSecOps práctico: cómo correr trivy, gitleaks y semgrep en PR sin frenar la velocidad del feature.

// aprendiendo

Aprendiendo

  • Profundizando en OpenTelemetry para observabilidad uniforme entre runtimes (Node, Python, Go) sin vendor lock-in.
  • Releyendo el WCAG 2.2 Understanding doc para afinar criterios AAA donde el costo es bajo (focus visible, target size).
  • Explorando eBPF aplicado a runtime security en clusters Kubernetes (Cilium Tetragon).
