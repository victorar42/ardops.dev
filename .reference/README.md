# Reference materials — NO PUBLICAR

Esta carpeta contiene materiales de referencia que NO forman parte del
sitio publicado. Su único propósito es servir como insumo para Copilot
y para decisiones de diseño.

## Contenido

### v1-design/index.html
Primera versión de la landing page de ardops.dev. Define:
- Paleta de colores (tema dark con acentos cyan #22d3ee y verde #4ade80)
- Tipografías (JetBrains Mono + Outfit)
- Componentes visuales: hero con grid + glow animado, talk card,
  pipeline de 7 etapas, stats cards, blog placeholder con cursor
  terminal, contact links
- Animaciones: fade-up escalonado, pulse-glow, terminal cursor blink
- Estética: terminal/code-first, grain overlay, monospace para acentos

**Esta versión es la fuente de verdad visual.** Cualquier rediseño debe
respetar paleta, tipografías, animaciones y estética terminal.
